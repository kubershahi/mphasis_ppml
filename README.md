# Mphasis PPML Research Project

## SecureML Implementation

### Dependencies
* Eigen
* emp-ot

### Execution
```
git clone https://github.com/kubershahi/mphasis_ppml.git
cd mphasis_ppml

// for linear regression
make linear
./linear

// for logitic regression
make logistic
./logistic

```
